# DataVine-Analytics-week8
Retail customer segmentation, product recommendation, and classification models built on e-commerce transactional data
## Retail Customer Analytics & Recommendation System
A machine learning project analyzing e-commerce transactional data to solve real-world retail business problems.
## Overview
Built as part of the Flatiron School AI & Data Science Bootcamp. Worked with customer purchase data across multiple regions to build predictive and analytical tools.
## What I Built

Customer Segmentation: Clustering analysis to uncover natural customer groups for marketing strategies
Classification Model: Predicted customer value segments from purchasing behavior
Recommendation System: Identified similar products using similarity metrics

## Tools & Libraries
Python, Pandas, NumPy, scikit-learn, Matplotlib, Seaborn
## Key Techniques
K-means clustering, PCA dimensionality reduction, hyperparameter tuning, feature engineering, model evaluation
